Overview

Purpose

The purpose of the Camping Trip Planner is to help users organize camping trips in one convenient place. Users can add trips, view important information such as destinations and dates, create packing checklists, and search through their planned adventures. The application is designed to make planning camping trips easier and more organized.

Audience

This website is intended for campers, outdoor enthusiasts, families, and anyone planning a camping trip. It is designed for users who want a simple way to organize trip details and packing lists before heading outdoors.

Dynamic elements

JavaScript will allow users to add camping trips. Each trip will be displayed dynamically as a card on the page. Users will be able to search for trips by name or destination, filter trips, mark packing list items as completed, and switch between light and dark mode. The page updates automatically in response to user interactions without needing to reload.

Content

Home page

The Home page welcomes users with a camping-themed hero image and displays their planned camping trips as interactive cards. Users can search for trips by destination or campground, filter trips based on their status, and switch between light and dark mode. Each trip card provides quick access to important trip information.

Images for the Home page

Add Trip page

The Add Trip page contains a form where users can enter details about a new camping trip, including the destination, travel dates, number of campers, camping type, and packing list. JavaScript validates the form before creating a new trip and displaying it on the Home page.

Pseudocode for your project

					Start program

					Create an array of camping trips.

					Display all camping trips on the Home page.

					When the user types in the search box
					Get the search text.
					Filter the trips array by trip name or destination.
					Display the matching trips.

					When the user clicks a filter button
					If "All" is selected
					Display every trip.
					Else if "Upcoming" is selected
					Display only upcoming trips.
					Else if "Past" is selected
					Display only past trips.

					When the user clicks the Add Trip button
					Open the Add Trip page.

					When the user submits the Add Trip form
					Check that all required fields are completed.
					If the form is valid
					Create a new trip object.
					Add the trip object to the trips array.
					Display the updated trip list.
					Else
					Display an error message.

					When the user clicks the Delete button
					Remove the selected trip from the array.
					Refresh the displayed trips.

					When the user clicks the Dark Mode button
					If Light Mode is active
					Change to Dark Mode.
					Else
					Change to Light Mode.

					End program
